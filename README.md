# Packfolio

A clean and minimal portfolio website.

## Features

- Responsive design
- Easy customization
- Fast performance

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/packfolio-clean.git
   ```
2. Navigate to the project directory:
   ```
   cd packfolio-clean
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage

Edit the files in the `src` directory to customize your portfolio.
